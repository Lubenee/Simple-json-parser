The size and capacity of this \doxylink{class_vector}{Vector} may change to match the size of the assigned \doxylink{class_vector}{Vector}. The assignment operator performs a deep copy of the elements, so each element is individually copied to the destination \doxylink{class_vector}{Vector}. This requires that the elements in have a defined operator=.

Reserves memory to accommodate a specified capacity for the \doxylink{class_vector}{Vector}. 

This method ensures that the \doxylink{class_vector}{Vector} has enough memory allocated to hold at least the specified number of elements. If the requested capacity is less than or equal to the current capacity, this method has no effect. Otherwise, it allocates additional memory to accommodate the requested capacity, potentially increasing the \doxylink{class_vector}{Vector}\textquotesingle{}s capacity.
